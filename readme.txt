== Description ==

Posts/Pages Edit SubPanel Date Format synchronize the wordpress date format with date format in date column of posts edit subpanel.

Currently when admin change the date format of website it does not reflect the changes on posts edit page in date column. This plugin solve this issue.

For more, visit: <a href="http://www.vizteck.com/blog/2010/06/17/wordpress-plugin-contribution-by-vizteck/">http://www.vizteck.com/bog</a> 

== Installation ==

1. Upload the folder "posts-edit-subpanel-date-format" to "/wp-content/plugins/"
1. Activate the plugin through the "Plugins" menu in WordPress
1. Done!
